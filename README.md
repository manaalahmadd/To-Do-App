📝 To-Do App

A simple and efficient To-Do List application to help you manage your daily tasks and stay organized. ✅

📌 About the Project

This is a web-based To-Do App built using HTML, CSS, and JavaScript.
It allows users to add, complete, and delete tasks easily.

🚀 Features
➕ Add new tasks
✅ Mark tasks as completed
❌ Delete tasks
💾 Saves tasks (if using local storage)
🎨 Simple and user-friendly interface

🛠️ Technologies Used
HTML
CSS
JavaScript
